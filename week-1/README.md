## Day 1: Meet the Turtle
1. Open Python (Thonny or Mu recommended).
2. Run this code:

```python
import turtle

t = turtle.Turtle()

t.forward(100)   # move forward 100 steps
t.right(90)      # turn right 90 degrees
t.forward(100)

turtle.done()
```
